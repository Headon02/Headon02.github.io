Milta Font

I made this font for my constructed language Milta. It is an abugida, which is achieved through the use of CV ligatures, e.g. <p + a = pa> would be p + a = pa. To make certain consonants in Milta easier to type, there are other ligatures. <bw, dw, gw, mw, lw> produce bw, dw, gw, mw, lw as oppposed to w following b, d, g, m, l. It is case sensitive, there are no ligatures for lower case followed by upper case, so <bA> is bA, but <ba, Ba, BA> are ba, Ba, BA.
